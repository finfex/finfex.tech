Projects

Gera

is an exchange rates generator. It collects exchange rates between currencies from stock exchanges. It helps to store and keep actual exchange rates for your app.

Features:

  • Fetching exchange rates from source each second
  • Building base exchange rates matrix for currencies using base or cross rates
  • Building exchange rates matrix for payment systems with a commission
  • Candlestick charts for base exchange rates by minutes
Check out the demo and the source code.
Bestchange

keeps you up to date about the exchange rate of any website on bestchange.ru.

Features:
